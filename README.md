# 📦 CLI Expenses Manager

A simple Python CLI (Command Line Interface) application for managing personal expenses. Users can add, view, calculate totals, and delete expense entries directly from the terminal.

## 📖 Project Description

This project is a lightweight expense tracking system built using Python. It demonstrates core backend fundamentals such as:

- Data handling with dictionaries
- Input validation
- Modular code structure
- CLI-based user interaction

The goal is to build strong foundational programming patterns while creating a usable tool.

## 🚀 Features

- ✅ Add expenses with validation
- ✅ View all stored expenses
- ✅ Calculate total spending
- ✅ Delete individual expenses
- ✅ Clear all expenses
- ✅ Continuous CLI loop for interaction
